## 💡 内容撰写公式

### Tagline公式
```
[身份] | [方法] + [价值]

示例：
"数据驱动的营销人 | 用洞察和创意连接用户与品牌"
```

### 能力卡片公式
```
图标 + 数字 + 标签
(hover) 详细说明

示例：
🎯 11组
深度访谈
(hover) 2000+条真实数据分析
```

### STAR公式
```
S: 背景 + 挑战 + 为什么重要
T: 角色 + 职责 + 团队
A: Step 1/2/3 + 方法
R: 量化 + 定性 + 截图
I: 学到 + 改进
```
